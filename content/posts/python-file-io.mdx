## Understanding File I/O in Python

File I/O in Python refers to the process of reading and writing data to and from files on the disk. Python provides built-in functions and methods to perform these operations seamlessly. Let's dive into some essential concepts and techniques for working with files in Python.

### Reading from a File

To read from a file in Python, you can use the `open()` function along with the `read()` method. Here's a simple example that reads from a text file named `sample.txt`:

```python
with open('sample.txt', 'r') as file:
    data = file.read()
    print(data)
```

In this code snippet:
- `open('sample.txt', 'r')` opens the file `sample.txt` in read mode.
- `file.read()` reads the contents of the file.
- `print(data)` displays the file's content.

### Writing to a File

Similarly, to write to a file in Python, you can use the `open()` function with the `write()` method. Let's see how you can write to a file named `output.txt`:

```python
with open('output.txt', 'w') as file:
    file.write('Hello, Python File I/O!')
```

Here, `open('output.txt', 'w')` opens `output.txt` in write mode, and `file.write('Hello, Python File I/O!')` writes the specified text to the file.

### Handling File Exceptions

When working with files, it's crucial to handle exceptions like `FileNotFoundError` and `PermissionError`. You can use `try-except` blocks to gracefully manage these errors. For example:

```python
try:
    with open('nonexistent.txt', 'r') as file:
        data = file.read()
except FileNotFoundError:
    print('File not found. Please check the file path.')
except PermissionError:
    print('Permission denied to access the file.')
```
