## What are Exceptions in Python?

In Python, an exception is an event that disrupts the normal flow of a program's execution. When an error occurs during runtime, Python raises an exception to indicate that something unexpected has happened. Examples of common exceptions include `ZeroDivisionError`, `TypeError`, `ValueError`, `FileNotFoundError`, and many more.

## Why Use Exception Handling?

Exception handling allows you to gracefully manage errors and unexpected situations in your code. By catching and handling exceptions, you can prevent your program from crashing and provide meaningful feedback to users when errors occur. This helps improve the overall reliability and robustness of your Python applications.

## How to Handle Exceptions in Python

### Try-Except Block

The most common way to handle exceptions in Python is by using a `try-except` block. Here's a simple example:

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Error: Division by zero")
```

In this code snippet, we attempt to divide 10 by 0, which would normally raise a `ZeroDivisionError`. However, by enclosing the risky code within a `try` block and catching the specific exception using an `except` block, we can handle the error gracefully.

### Finally Block

You can also use a `finally` block to execute cleanup code regardless of whether an exception is raised or not. This is useful for releasing resources or closing files, for example:

```python
try:
    file = open("example.txt", "r")
    # Perform file operations
except FileNotFoundError:
    print("Error: File not found")
finally:
    file.close()
```

### Custom Exceptions

In addition to built-in exceptions, you can create your custom exceptions by defining a new class that inherits from the `Exception` class. This allows you to raise and handle custom exceptions tailored to your specific application logic.

```python
class CustomError(Exception):
    def __init__(self, message):
        self.message = message

try:
    raise CustomError("Custom error message")
except CustomError as e:
    print(f"Custom Error: {e.message}")
```
