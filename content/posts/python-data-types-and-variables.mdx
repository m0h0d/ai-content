## Data Types in Python

Python has several built-in data types to store and manipulate different kinds of data. Some of the commonly used data types include:

1. **Integer**: Represents whole numbers.
2. **Float**: Represents decimal numbers.
3. **String**: Represents text.
4. **Boolean**: Represents True or False values.
5. **List**: Represents an ordered collection of items.
6. **Tuple**: Represents an ordered, immutable collection of items.
7. **Dictionary**: Represents a collection of key-value pairs.
8. **Set**: Represents an unordered collection of unique items.

Each data type has specific characteristics and uses in Python programming. Let's dive into some examples to understand them better.

### Variables in Python

Variables are used to store data values in Python. You can assign a value to a variable using the assignment operator `=`. Variables can store different types of data, and their values can be changed during the program execution.

Here are some examples of defining variables in Python:

```python
# Integer variable
age = 25

# Float variable
height = 5.9

# String variable
name = "Alice"

# Boolean variable
is_student = True
```

### Code Examples

#### Lists in Python

Lists are versatile data structures in Python that can store multiple items. They are defined using square brackets `[]`.

```python
# Creating a list
numbers = [1, 2, 3, 4, 5]

# Accessing elements in a list
print(numbers[2])  # Output: 3

# Modifying elements in a list
numbers[0] = 10
print(numbers)  # Output: [10, 2, 3, 4, 5]
```

#### Dictionaries in Python

Dictionaries store key-value pairs and are defined using curly braces `{}`.

```python
# Creating a dictionary
person = {'name': 'John', 'age': 30, 'city': 'New York'}

# Accessing values in a dictionary
print(person['age'])  # Output: 30

# Modifying values in a dictionary
person['city'] = 'Chicago'
print(person)  # Output: {'name': 'John', 'age': 30, 'city': 'Chicago'}
```
