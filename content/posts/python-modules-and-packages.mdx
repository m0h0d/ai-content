## Understanding Python Modules

In Python, a module is a file that contains Python code. Modules can define functions, classes, and variables that can be used in other Python files. To use a module in your code, you need to import it using the `import` statement.

For example, let's create a simple module named `math_operations.py`:

```python
# math_operations.py

def add(x, y):
    return x + y

def subtract(x, y):
    return x - y
```

To use the functions defined in the `math_operations` module, you can import it in another Python file like this:

```python
import math_operations

result = math_operations.add(5, 3)
print(result)  # Output: 8
```

## Working with Python Packages

A package in Python is a way of organizing related modules into a directory hierarchy. A package can contain subpackages, modules, and even other packages. To create a package, you need to create a directory with an `__init__.py` file inside it.

For example, let's create a package named `shapes` with two modules `circle.py` and `rectangle.py`:

```python
# shapes/circle.py

import math

def calculate_area(radius):
    return math.pi * radius ** 2
```

```python
# shapes/rectangle.py

def calculate_area(length, width):
    return length * width
```

To use the `shapes` package in your code, you can import specific modules like this:

```python
from shapes import circle, rectangle

circle_area = circle.calculate_area(5)
print(circle_area)  # Output: 78.54

rectangle_area = rectangle.calculate_area(4, 6)
print(rectangle_area)  # Output: 24
```
